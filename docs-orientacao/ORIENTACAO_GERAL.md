# Orientacao Geral

## Objetivo
Manter o projeto consistente com as preferencias do autor, garantindo qualidade tecnica e evolucao organizada.

## Stack principal (projeto atual)
- Frontend: Next.js (App Router) + TypeScript + TailwindCSS.
- Bibliotecas relevantes: Radix UI, framer-motion, next-i18next.
- Deploy: OpenNext/Cloudflare (scripts `build:cloudflare` e `deploy:cloudflare`).

## Idiomas e convencoes
- Codigo: ingles.
- Documentacao: pt-BR (sem acentos quando possivel).
- Logs: pt-BR, estruturados e objetivos.

## Principios gerais
- Preferir componentes funcionais e reutilizaveis.
- Separar UI de logica (hooks, services, lib).
- Manter design consistente com o projeto existente.
- Evitar complexidade desnecessaria e duplicacao.

## Documentacao obrigatoria
- Endpoints (quando existirem) devem ter documentacao em OpenAPI/Swagger em pt-BR.
- Explicar solucoes tecnicas com referencias oficiais quando relevante.

## Observacoes
- Este repositorio e um portfolio Next.js. Regras de backend Java ficam no arquivo de referencia.
